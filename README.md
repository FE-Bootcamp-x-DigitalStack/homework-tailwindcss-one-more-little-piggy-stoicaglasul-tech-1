# TailwindCSS assignment

Clone the piggy from the image with the help of TailwindCSS utilities.

Add the required HTML and TailwindCSS classes to replicate it as close as you can.

Use the following HEX codes for colors:

- #FFFFFF
- #F9BECA
- #000000
- #F46E86

Check the documentation for TailwindCSS utilities: https://tailwindcss.com/docs/styling-with-utility-classes
and refer to the class workshop for how to quickly integrate TailwindCSS.

For this assignment, the `index.html` file already has the Tailwind Play CDN link referenced.

- Use the `<style>` element from `index.html` which has the attribute of `text="type/tailwindcss"` to write all the Tailwind CSS code (like `@theme`, `@layer base`, `@layer components`).
- Use the `style.css` file to write all custom CSS (standard CSS syntax, not Tailwind).
